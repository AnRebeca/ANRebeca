#### Projeto SQL Escola com PostgreSQL 
- **Descrição:** Este projeto simula o banco de dados de uma escola , contendo informações de alunos, materiais e notas. O objetivo é praticar a modelagem de dados, criação de tabelas, inserção de registros e consultas SQL utilizando o PostgreSQL.
